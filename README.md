hello! chirpy-cli is a CLI for finding/listening to bird sounds from around the world, using the xeno-canto database. 

it does 3 things: 
1. search a bird by name. you can hear it sing and information about its recording
2. find birds near me. type in a location and see information about birds near that location
3. morning chorus. pick a place and hear local birds sing! 

You will need a mac (cuz it uses afplay, which mac uses) and a xeno-canto APi key (its free!). You might also need to make your own .env file :)

Importantly, english names only! spelling is important! Kirtland's Warbler is good, but Kirtlands Warbler is not. Apostrophies are important. 

also importantly, all the sounds come from volunteers all over the world who upload to xeno-canto. Thanks to those people!
